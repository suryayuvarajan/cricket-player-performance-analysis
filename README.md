# T20 Player Performance Analysis – Kohli vs Buttler

**Project Overview**
This project analyzes and compares the T20 batting performance of Virat Kohli and Jos Buttler using ball-by-ball data. The goal is to identify differences in scoring patterns, phase performance, and dismissal trends.

**Tools Used**
- Excel (Data Cleaning)
- Python (Pandas for analysis)
- Power BI (Dashboard & visualization)

**Key Features**
- Phase-wise strike rate analysis (Powerplay, Middle, Death)
- Run distribution comparison
- Boundary percentage analysis
- Dismissal pattern insights

**Key Insights**
- Virat Kohli shows stronger Powerplay intent, while Jos Buttler dominates in the death overs.
- Buttler relies more on six-hitting, whereas Kohli scores through consistent boundary accumulation.
- Kohli’s higher caught dismissals suggest increased risk during acceleration phases.
